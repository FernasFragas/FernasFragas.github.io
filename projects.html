Synapse Platform

A high-performance event ingestion and processing platform built in Go. It consumes real-time event streams from Kafka, normalizes heterogeneous data into strongly typed domain events, and exposes them through a secure JWT-authenticated REST API. Designed with scalability, resilience, and observability in mind, the platform includes OpenTelemetry tracing, Prometheus metrics, structured logging with sensitive-data redaction, dead-letter queue handling, graceful shutdown orchestration, and rate limiting. Built using domain-driven design principles, dependency injection, SQLite persistence, and type-safe database access with sqlc. The system was built to emphasize reliability, maintainability, and operational visibility in distributed event-driven systems.
